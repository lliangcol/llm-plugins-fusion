# Workflow Evaluations

These public-safe fixtures test canonical workflow relationships without claiming model-quality equivalence.

- `route/cases.json` contains deterministic conformance cases for routing inventory, ownership, packs, required inputs, and write boundaries.
- `surface-ab.json` is generated by `node scripts/evaluate-workflow-surfaces.mjs --write`. It compares static direct-command and compatibility-contract costs; it is not a live latency or model-quality result.
- Live assistant evidence belongs under `evidence/` and must name the assistant, exact version, execution mode, commit, result, and skipped capabilities.

Run `npm run eval:route` and `npm run eval:surfaces`. Never add private consumer prompts, credentials, endpoints, or repository names.
