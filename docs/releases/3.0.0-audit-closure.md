## Intentional exception

The pre-write hook retains a thin Bash launcher. A live Claude Code 2.1.205
probe showed that a missing direct Node command is non-blocking and therefore
fail-open; the launcher preserves fail-closed behavior when Node.js is absent.
Post-use audit events use direct Node exec form. This exception is tested and
documented rather than reported as a completed Bash removal.

## Publication completion conditions

The signed immutable `v3.0.0` tag completed all gates through release evidence,
but its final attestation input used an unsupported glob and no GitHub Release
was published. Completion therefore requires all required checks, signed
annotated `v3.0.1` creation, immutable tag
ruleset readback, exact-tag isolated install and OAuth route smoke, artifact
attestation, GitHub Release success, and post-publication digest comparison.
