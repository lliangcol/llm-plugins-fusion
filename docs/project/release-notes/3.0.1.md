# 3.0.1 Release Notes

Status: prepared; publication requires all release gates

Version 3.0.1 contains the complete 3.0.0 audit-hardening work and fixes the
final attestation input contract. The release workflow now resolves exactly
one deterministic archive and exactly one CycloneDX SBOM, validates both
counts, and passes their exact paths to `actions/attest`.

The signed `v3.0.0` tag is intentionally unchanged. Its trusted preflight,
pre-release validation, exact-tag Claude install, OAuth route smoke, and
release-evidence aggregation passed; the final attestation job failed before a
GitHub Release was created. This patch is the immutable recovery release.
