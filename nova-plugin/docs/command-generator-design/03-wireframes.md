# 界面草图（Wireframes）

> 说明：以下为低保真草图，用于对齐页面结构与交互，不代表最终视觉设计。

## 1) 场景首页（移动端优先）

```text
┌──────────────────────────────────────────┐
│ 搜索：🔎 需求分析/排障/评审/实现...        │
├──────────────────────────────────────────┤
│ 场景分类                                  │
│ 需求分析                                  │
│  - 新功能需求分析  → /senior-explore      │
│  - 快速对齐认知    → /explore-lite        │
│ 故障排查                                  │
│  - 生产问题深挖    → 工作流 B             │
│ ...                                       │
├──────────────────────────────────────────┤
│ 最近使用                                  │
│  - /produce-plan  2026-01-xx              │
│  - /plan-lite     2026-01-xx              │
└──────────────────────────────────────────┘
底部 Tab：场景 | 命令 | 工作流 | 历史 | 设置
```

## 3) 单命令生成（桌面三栏 / 移动单栏）

### 桌面（示意）

```text
┌────────────────────────────────────────────────────────────────────────────┐
│ TopBar: [Search]   命令：/produce-plan   [Examples] [History] [Export]      │
└────────────────────────────────────────────────────────────────────────────┘
┌───────────────┬────────────────────────────────┬───────────────────────────┐
│ 左：字段导航   │ 中：表单                        │ 右：预览/输出               │
│ - 必填(2)      │ PLAN_OUTPUT_PATH*  [选择/输入]  │ 预览（可编辑）               │
│ - 可选(2)      │ PLAN_INTENT*      [多行]        │ ─────────────────────────  │
│ - 文件(1)      │ ANALYSIS_INPUTS   [+ 添加]      │ /produce-plan               │
│               │ CONSTRAINTS       [+ 添加]      │ PLAN_OUTPUT_PATH: ...       │
│               │                                  │ ...                         │
│               │ [保存到应用内] [生成并复制]      │ [复制] [导出]               │
└───────────────┴────────────────────────────────┴───────────────────────────┘
```

### 移动端（示意）

```text
┌──────────────────────────────────────────┐
│ /produce-plan   [Examples]                │
├──────────────────────────────────────────┤
│ 字段（折叠面板）                           │
│ ▸ 必填：PLAN_OUTPUT_PATH*  [输入]          │
│ ▸ 必填：PLAN_INTENT*       [多行]          │
│ ▸ 可选：ANALYSIS_INPUTS    [+]             │
│ ▸ 可选：CONSTRAINTS        [+]             │
├──────────────────────────────────────────┤
│ 文件插入（可选）                           │
│ [+ 上传文件]  策略：路径/片段/全文         │
├──────────────────────────────────────────┤
│ 预览（可编辑）                             │
│ /produce-plan                              │
│ PLAN_OUTPUT_PATH: ...                      │
│ ...                                        │
├──────────────────────────────────────────┤
│ [复制] [保存到应用内] [下载] [分享]        │
└──────────────────────────────────────────┘
```

## 4) 工作流向导（Stepper）

```text
┌──────────────────────────────────────────┐
│ 工作流：新需求开发（需求不清晰）           │
├──────────────────────────────────────────┤
│ Step 1 /senior-explore    [进行中]         │
│ Step 2 /plan-lite         [待开始]         │
│ Step 3 /produce-plan      [可选]           │
│ Step 4 /plan-review       [待开始]         │
│ Step 5 /implement-standard [待开始]        │
│ Step 6 /finalize-work     [待开始]         │
├──────────────────────────────────────────┤
│ 当前步骤：生成面板（同单命令）             │
│ 变量：plan_output_path=...                 │
│      analysis_export_path=...              │
│ [跳过此步] [生成并保存] [下一步]           │
└──────────────────────────────────────────┘
```
