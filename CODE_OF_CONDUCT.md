# Contributor Covenant Code of Conduct

## 我们的承诺

我们作为成员、贡献者与维护者，承诺让每一位参与本社区的人，不论年龄、体型、可见或不可见的身体状况、族群、性别特征、性别认同与表达、经验水平、受教育程度、社会经济地位、国籍、外貌、种族、宗教信仰、性取向，都能获得无骚扰的体验。

我们承诺以有助于开放、友善、多元、包容、健康社区的方式行事与互动。

## 我们的准则

有助于营造良好社区氛围的行为包括：

- 对他人展现同理心与善意
- 尊重不同的观点与经验
- 给出与接受建设性反馈
- 为错误承担责任、向受影响者道歉、并从经验中学习
- 关注对整个社区最有利的决定，而不只是个人利益

不可接受的行为包括：

- 使用带有性意味的语言或图像，以及任何形式的性关注或挑逗
- 挑衅、侮辱、贬损性言论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，公开他人私人信息，如物理地址或电子邮箱
- 在专业场合中合理认为不恰当的其他行为

## 执行责任

社区维护者负责解释并执行我们的行为准则，并会对任何被认为不合适、具有威胁性、冒犯或有害的行为采取恰当且公平的纠正措施。

社区维护者有权并有责任移除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki 编辑、issue 与其他贡献，并在合适时说明理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公开场合代表社区时的行为。代表社区的例子包括使用官方邮箱、通过官方社媒账号发帖、或在线上/线下活动中作为指定代表出席。

## 举报

冒犯、骚扰或其他不可接受的行为可通过以下方式举报：

- Email: `lliangcoder@gmail.com`

所有投诉都会被及时、公平地审阅与调查。所有社区维护者有义务尊重报告者的隐私与安全。

## 执行指南

社区维护者将遵循以下社区影响指南，来判定对违反本行为准则的行为所采取的后果：

### 1. 更正

**社区影响**：使用不恰当的语言，或在社区中被认为不专业或不受欢迎的其他行为。

**后果**：由社区维护者私下以书面形式发出警告，说明违规性质并解释其不当之处。可能要求公开致歉。

### 2. 警告

**社区影响**：一次性违规，或一系列违规行为。

**后果**：警告并说明后续违规将引发的后果。在指定时间内不得与相关人员主动互动，包括不得主动与执行行为准则的人员互动。违反此条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违规，包括持续不当行为。

**后果**：临时禁止以任何形式与社区互动或公开交流。在此期间不得与相关人员主动互动，包括不得主动与执行行为准则的人员互动。违反此条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：持续违规，包括持续不当行为、骚扰个人、或对某群体的攻击或贬低。

**后果**：永久禁止以任何形式在社区内公开互动。
